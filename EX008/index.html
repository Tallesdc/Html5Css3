Principais Formatações

Negrito / Destaque

Nesta frase um termo termo em negritousando a tag <b>(não tem semântica).

Nesta frase, tenho um termo em destaqueusando a tag <strong>(semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>.

E no outro parágrafo, temos outro texto marcado no final.

Para colocar a cor desejada na tag <mark> basta utilizar o paramentro style dentro da tag junto com o background-color.

Se você tiver vários mark's na pagina o ideal é utilizar a tag <style> dentro da parte 'head' da configuração da sua página.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. A tag <big> vai deletada porém, a tag <small> permanece.

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado. Antigamente a tag utilizada era a <strike> e na versão atual utilizamos a tag <dell>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Neste caso a tag utilizada é a <ins>.

Existem também o sublinhado com a tag não semântica <u>.

Texto sobrescrito

Para inserir coisas do tipo X2+3.

Para este caso deve-se utilizar a tag <sup>.

Texto Subscrito

Para Inserir coisas como H2O.

Para este caso deve-se utilizar a tag <sub>.

Trechos de código

Exemplo de código em Python

num = input(input('Digite um número')) if num % 2 == 0: print(f'0 número {num} é par') else: print(f'0 número {num} é impar') print('Fim Do Programa')

Ao usar a tag <code> para inserir trechos de código como o que foi feito no texto acima. Porém, o texto não vai aparecer com a formatação adequada.

Neste caso para inserir o trecho de código com a devida formatação deve-se inserir a tag <pre> junto a tag <code> sendo assim o trecho ficará com a formatação adequada conforme esta demonstrado logo a seguir:

 num = input(input('Digite um número')) if num % 2 == 0:
            print(f'0 número {num} é par')
            else:
            print(f'0 número {num} é impar')
            print('Fim Do Programa')

"Citações"

Para criar uma citação podemos usar a tag <q> (do inglês quote, que significa citar). O texto que estiver estre esta tag já vai aparecer automaticamente entre as apas.

Exemplo: Tags unidas, jamais serão vencidas!

Também podemos criar citações mais longas (em bloco) que tenham um parágrafo só para si. Nesse caso, sera utilzada a tag <blockquote>.

Exemplo:

O PHP é uma linguagem para criação de scripts para a Web do lado servidor embutidos em HTML, cujo código-font é aberto, e que é compativel com os mais importantes servidores Web (principalmebte Apache).

Também podemos colocar um link para o texto original usando o parâmetro cite dentro da tag.

Abreviações

Sempre que quiser escrever uma sigla, mas deixar claro para o usuário ( e aos mecanismos de busca) o significado dela, use a tag <abbr>.
