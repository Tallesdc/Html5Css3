Incluir Imagens e Favicon.

Vamos Começar Imagens em PNG.

Para incluir as imagens em nosso site vamos utilizar a tag <img>.

Antes de incluir a imagem deve-se ficar atento ao seu tamanho. Imagens grandes podem demorar a abrir no site e proporcionar ao usuário uma experiência desagradável.
E vale a pena lembrar que o Google não recomenda sites com este tipo de situação.

Para incluir a imagem basta digitar a tag <img> e logo após acionar o comando CTRL + Space entre as aspas que estão no parametro SRC para selecionar a imagem desejada. Já no parametro alt deve-se colocar a descrição da imagem selecionada.

Muito Chiq! 🖖

Para incluirmos o Favicon é utlizada a tag <link.favicon> partir daí é utilizado o mesmo procedimento para inclusão de imagens.

Neste caso deve-se ficar atento ao formato da imagem deve ser .ico.
