Revisão

Parágrafos e quebras de linha!

Vamos recapitular o que foi vista até agora no curso!

Vimos que para inserir um paragrafo é necéssario utilizar a tag <p> e que para realizar quebra de linha será necessário utilizar a tag <br>.

Hierarquia de titulos

Os titulos seguem uma hierarquia que vai de <h1> até <h6>.

Inclusão de Simbolos e Emojois

Vamos fazer a inclusõa de simbolos e emojisn (para ter acesso as tags de emojis basta acessar o site "https://www.w3schools.com/charsets/ref_emoji.asp"). Atravez dele você terá acesso a varias tag's refrente a emojis e simbolos e para incluir basta digitar a tag da seguinte maneira: '& junto com o comando #129409' assim você terá a tag do Leão 🦁, Joystick 🎮, Vaca 🐄, Bufalo 🐃.

Inculão de Imagens

Vamos fazer inclusões de imagens.

Atenção - Antes de inserir as imagens deve-se se atentar aos direitos reservados a imagem.

🤓 Quanto mais facil for para imagem ser carregadas no site sera melhor ou seja é sempre bom editar as imagens antes de inserir no site.

🤓 Você também pode inserir uma imagem copiando a url da mesma direto do site.

🤓 A tag utlizada para inclusão de imagens é <img>.
